Data Cleaning Method

Please select which base data file you would like to use to create your customized data set. There are two base data files to choose from: baseline and modal. The baseline data file includes all eye movement data with no trimming. The modal data file includes data trimming using the most common methods used to clean eye movement data: merging fixations that are shorter than 80ms with longer fixations within .50 degree of visual angle and deleting fixations that are shorter than 80ms or longer than 800ms. The user might choose additional data cleaning methods such as removing outliers. For more information about best practices for cleaning eye movement data, see Eskenazi (2023) here.

Identifying Variables

These variables are used to build the basic structure of the data file such as Participant ID, Passage ID, and Word. Participant ID and Passage ID should always automatically be checked by default.

Trial Data

This section includes data about the entire trial, not individual interest areas.

Interest Area size

This section includes information about pixel size and position of the interest area.

Total Reading Time

This section includes total reading time in the interest area.

First Fixation Data

This section includes information about the first fixation that was made in the interest area.

First Run Data

This section includes information about the first time that the participants’ eyes entered the interest area until when the participants’ eyes first left the interest area.

Fixation Data

This section includes information about the number and types of fixation made in the interest area.

Regression Data

This section includes information about regressive eye movements.

Other Eye Movement Data

This section includes other eye movement behaviors such as word skipping

Individual Difference

In this section, you can select any variable related to individual differences in reading skill or demographic traits of the participant that was collected during the reading skill battery portion of this study.

Vocabulary

This section includes individual differences in vocabulary skill measured by the Nelson Denny Reading Test Form I.

Comprehension

This section includes individual differences in comprehension skill measured by the Nelson Denny Reading Test Form I..

General Reading Ability

This section includes individual differences in general reading ability measured by the Nelson Denny Reading Test Form I..

Working Memory

This section includes individual differences in working memory as measured by the operation span task.

Phonological Processing

This section includes individual differences in phonological processing as measured by the Comprehension Test of Phonological Processing (CTOPP).

Spelling

This section includes individual differences in spelling ability as measured by spelling dictation and spelling recognition tasks (Eskenazi, 2022).

Morphology

This section includes individual differences in morphological processing as measured by two tasks ( Bernstein et al., 2020 ; Carlisle, 2000 ).

Demographics

This section includes demographic variables for each participant.
